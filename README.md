# seg_counter

7-segment display counter using ATmega328p

Based on [this guide](https://arduinointro.com/articles/projects/how-to-use-a-7-segment-display-with-arduino-a-complete-beginners-guide),
but simplified to use AVR library directly.

[See video](https://www.youtube.com/shorts/0S_yj877N4Y) of how it works.

> In the updated version segment was moved "a" to PB2,
> because PD2 is used for INT0.
